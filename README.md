# Excel Data Analysis Dashboard

## Project Overview
This project presents an interactive Excel dashboard for analyzing global data jobs, salaries, employment types, job roles, and hiring trends.

## Tools Used
- Microsoft Excel
- Charts
- Maps
- Data Validation
- Formulas

## Dashboard Features
- Interactive Job Title filter
- World Map visualization
- Median Salary analysis
- Job Count summary
- Employment Type analysis
- Top Job Platform insights

## Process
- Collected and cleaned the dataset.
- Applied formulas and data validation.
- Built interactive charts and map visualizations.
- Created a dashboard for easy analysis and decision-making.
